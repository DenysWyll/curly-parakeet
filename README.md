# README 1.0
Esta aplicação roda com a versão 4.0 do Node.js

## Configuração
Instalar o node.js 4.x 
Instalar o mongodb 3.2.x

Deixar o servidor mongo rodando 

    $ mongod

Criar a base de dados 


    $ mongo
    $ use denys_teste;

Iniciar aplicação


    $ node app.js

O sistema estará rodando na porta 8080


    http://localhost:8080
